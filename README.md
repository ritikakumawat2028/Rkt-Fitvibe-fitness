🏋️‍♂️ RKT-FitVibe – Fitness & Training Platform

RKT-FitVibe is a full-stack fitness web application built using React, JavaScript, Node.js, MongoDB, and CSS.
It provides a complete fitness management system with user registration, login, dashboards, admin panel, trainers, workout plans, and exercise guidance.

🚀 Features
👤 User Features

User Registration & Login (Authentication)

Secure User Dashboard

View workout & exercise plans

Choose fitness plans

Trainer assignment & details

Responsive UI for all devices

🛠️ Admin Panel Features

Admin Authentication

Manage users (view, update, delete)

Add / update / remove trainers

Manage workout plans

Manage exercise content

Full control over website data

🧑‍🏫 Trainer Management

Add trainers with specialization

Assign trainers to users

Manage training programs

🧱 Tech Stack
Frontend

React.js

JavaScript (ES6+)

CSS

React Router

Axios

Backend

Node.js

Express.js

RESTful APIs

Database

MongoDB

Mongoose ODM

Authentication

JWT (JSON Web Token)

Role-based access (Admin / User)
